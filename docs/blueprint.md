# **App Name**: LinguaLearn

## Core Features:

- Question Display: Display a question in the target language along with four answer choices.
- Answer Selection: Allow the user to select one of the four provided answers.
- Score Tracking: Immediately update and display the user's score after each answer.
- Repetitive Learning: Cycle through the questions repeatedly to reinforce learning.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) to evoke a sense of calm and focus, fitting for a learning environment.
- Background color: Very light blue (#F0F8FF) to keep the focus on the question and answers, offering a clean and unobtrusive backdrop.
- Accent color: Soft orange (#FFB347), creating a warm, friendly contrast to signal interaction and progression.
- Body and headline font: 'PT Sans', a humanist sans-serif which combines a modern look and a little warmth or personality; suitable for headlines or body text
- Use clear and simple icons to represent different language categories or concepts.
- Present questions and answers in a clear, uncluttered layout, with a focus on readability.
- Use subtle animations for correct answer feedback and score updates.
